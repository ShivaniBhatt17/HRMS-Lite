## HRMS Lite

### Overview
A lightweight HRMS to manage employees and attendance.

### Tech Stack
- Django + DRF
- React
- PostgreSQL

### Features
- Add/Delete Employees
- Mark Attendance
- View Attendance

### Setup
Backend:
- pip install -r requirements.txt
- python manage.py migrate
- python manage.py runserver

Frontend:
- npm install
- npm start
